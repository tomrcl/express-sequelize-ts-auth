# express-sequelize-ts-auth

`npm install`  
`npm start`

## Exemple .env file

```
DB_HOST=localhost
DB_PORT=6543
DB_USER=postgres
DB_PASSWORD=admin
DB_NAME=OMACAP
DB_DRIVER=postgres

APP_PORT=3333

APP_ACCESS_TOKEN_SECRET=secretToken
APP_ACCESS_TOKEN_EXP=20m
APP_REFRESH_TOKEN_SECRET=refreshSecretToken
APP_REFRESH_TOKEN_EXP_DAYS=10
```
